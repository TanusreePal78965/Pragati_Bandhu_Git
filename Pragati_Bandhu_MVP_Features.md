# 📱 Pragati_Bandhu – MVP Feature Checklist

## 🎯 Core Principle

Offline-first billing app for small Indian shopkeepers.
Bills must be saved even without internet.

---

## ✅ INCLUDED IN MVP (v1)

### 🔐 Authentication

- [ ] Phone number login
- [ ] OTP verification
- [ ] Auto login on next app open

### 🏪 Shop Setup

- [ ] Shop name setup (first time only)
- [ ] Owner name setup
- [ ] Single shop per user

### 📦 Product Management

- [ ] Add product (name, price)
- [ ] Edit product
- [ ] Delete product
- [ ] Optional stock quantity
- [ ] Products stored locally (offline)

### 🧾 Billing

- [ ] Create new bill
- [ ] Add multiple products to bill
- [ ] Quantity selection
- [ ] Auto total calculation
- [ ] Save bill offline
- [ ] Generate local bill ID

### 🤝 Udhar (Credit)

- [ ] Add customer
- [ ] Assign bill to customer
- [ ] Track pending amount
- [ ] Partial payment entry
- [ ] Update balance locally

### 🔄 Offline & Sync

- [ ] Local database (SQLite)
- [ ] Sync queue for unsynced data
- [ ] Auto sync when internet is available
- [ ] Sync status indicator

### 📊 Reports (Basic)

- [ ] Daily sales total
- [ ] Total cash received
- [ ] Total udhar pending
- [ ] Works offline

### ⚙️ Settings

- [ ] Logout
- [ ] Language selection (English / Hindi)
- [ ] Basic app info

---

## ❌ NOT INCLUDED IN MVP (Future Scope)

- [ ] GST calculation
- [ ] Invoice PDF / printing
- [ ] Multiple shops
- [ ] Staff / employee accounts
- [ ] Advanced analytics & charts
- [ ] Online payments inside app
- [ ] Customer WhatsApp automation
- [ ] Cloud-only usage (offline is mandatory)

---

## 🛑 MVP LOCK RULE

Any feature not listed above is **NOT allowed** in v1.

Changes only after:
- Real user testing
- Data shows strong need
