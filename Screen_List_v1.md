# 📱 Pragati_Bandhu – Screen List (v1)

## 🎯 Purpose

This document defines the **final list of screens** for VyapariSathi MVP.
No new screens should be added in v1.

---

## 🧭 App Screens (Final)

### 1. Splash / Loading Screen

- App logo
- App initialization
- Auth + local DB check
- Sync status check

---

### 2. Login Screen

- Phone number input
- Continue button
- Simple, distraction-free UI

---

### 3. OTP Verification Screen

- OTP input
- Auto-read (if possible)
- Verify button
- Resend OTP option

---

### 4. Shop Setup Screen (First-Time Only)

- Shop name input
- Owner name input
- Save & continue
- Shown only once after first login

---

### 5. Home Dashboard

- Today’s total sales
- Pending udhar amount
- Primary actions:
  - New Bill
  - Products
  - Customers
- Sync status indicator

---

### 6. New Bill Screen

- Product search / list
- Quantity selector
- Item-wise total
- Bill total
- Save bill button (offline allowed)

---

### 7. Product Management Screen

- Product list
- Add new product
- Edit product
- Delete product

---

### 8. Customer (Udhar) Screen

- Customer list
- Pending amount per customer
- Add new customer
- Customer detail view

---

### 9. Payment Screen

- Add payment for customer
- Partial payment support
- Update pending balance
- Payment history (basic)

---

### 10. Reports Screen (Basic)

- Daily sales total
- Cash collected
- Udhar pending
- Offline-first data

---

### 11. Settings Screen

- Language selection
- Logout
- App info
- Basic preferences

---

## 🛑 Screen Lock Rule

- No additional screens in MVP v1
- Any new screen goes to v2 backlog
- Simplicity > completeness
